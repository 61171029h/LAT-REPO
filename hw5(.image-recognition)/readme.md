<h2>➲主要應用於偵測所做的簡報是否符合多媒體原則(顏色小於三種)  ✍🏻</h2>
<ul>
  <a>基於多媒體原則，若簡報多於三種顏色，內容將會難以觀看，因此做出此分析，</a>
  <li><a>若顏色小於三種，給予稱讚</a></li>
  <li><a>若顏色大於三種，建議修改</a></li>
</ul>
